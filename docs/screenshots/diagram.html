How Mailgraph works

From Postfix logs to browser charts — no Apache, no Perl

📬

1. Mail server

  • Postfix sends and receives mail
  • Writes syslog lines
  • File: mail.log
▶
1 Go binary
⚙️

2. Mailgraph

  • Reads the log in real time
  • Counts sent, SPF, spam…
  • Stores data in RRD files
▶
💾

3. History (RRD)

  • One-minute buckets
  • Legacy Mailgraph compatible
  • Survives restarts
▶
📊

4. Browser

  • Interactive charts
  • /today, /last-week…
  • Refreshes every 5 min
